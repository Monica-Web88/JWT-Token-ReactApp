# JWT Token in React Login Dashboard

## Tech Stack
- Frontend: React
- Backend: Node.js
- Authentication: JWT Token

## Features of the Application
- Demonstrates a login dashboard using JWT-based authentication (login, refresh, and logout)
- Utilizes a Node.js backend to handle API requests
- Detects expired tokens and securely requests new access tokens using refresh tokens
- Provides admin functionality to manage users, including the ability to delete users
