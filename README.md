# JWT Token in React Login Dashboard

Frontend - React
Backend - Node.js
Authentication - JWT Token

##Features of the Applicaiton
--Demo of a Login Dashboard with JWT Token authentication during User Login, Refresh and Logout.
--Uses a Node.js backend server for handling the API requests
--Check if token has expired and request a new refresh token by passing along the access token to the backend Node.js server.
--Admin User has the option to Delete User
